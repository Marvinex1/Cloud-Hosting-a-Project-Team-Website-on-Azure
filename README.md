# Cloud-Hosting-a-Website-on-Azure
A step by step guide to hosting a website on Azure Cloud
## Step 1- Resource group
	• Create an Azure account
	• Create a resource group
## Step 2- Storage
	• Go to storage account to create storage
	• Select the resource group for the storage being created
	• Provide instance details and storage type
	• Set performance and redundancy as required standard or premium and LRS for low cost or GZRS for optimal data protection
	• Create and deploy
## Step 3
	• Go to Data management and click static website or the resource created and 'enable'
	• Enter index document name as 'index.html' and error doc path as '404.html' and save
	• A weblink is generated as the primary endpoint
## Step 4- Container
	• Go to Data storage and click 'containers'
	• Click web
	• 'upload' the html files downloaded or web script for any website or free templates online
## Step 5
Copy the weblink (primary endpoint) and test launch
